# Đối với người trả lời
- Câu hỏi dài quá thì lười đọc
- Câu hỏi ngắn quá thì không thấy đủ kích thích
- Trả lời ngắn quá thì người hỏi không hiểu
- Trả lời dài quá thì tốn thời gian của mình
